## 🚀 本地开发

### 1. 安装依赖
无需 npm，微信小程序原生开发无需额外依赖

### 2. 配置 appid
编辑 `project.config.json`，将 `appid` 替换为你的小程序 appid：
```json
"appid": "wxxxxxxxxx"
```

### 3. 打开项目
使用**微信开发者工具**，导入 `miniprogram/` 目录

### 4. 调试
在微信开发者工具中：
- 编译模式 → 预加载所有页面
- 勾选"不校验合法域名"（开发阶段）
- 使用"真机调试"测试相机/相册等 API
